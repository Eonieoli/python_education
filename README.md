- **일일 시간**: 8교시 (50분 × 8)

## 📁 디렉토리 구조

```
python_education/
```

## 📄 파일 구조

### 일반 교시 (2-7교시)
- `0_notebook.ipynb` - 개념 설명 + Basic 코드
- `1_basic.py` - 강사 시연용 예제
- `2_practice.py` - TODO 포함, 함께 따라하기
- `3_exercise.py` - 스스로 푸는 문제
- `solution/3_exercise_solution.py` - 정답

### 8교시 종합 실습
- `0_notebook.ipynb` - 복습 퀴즈 + 문제 개요
- `1_basic.py` - 🟢 기초 Problem (15분)
- `2_intermediate.py` - 🟡 응용 Problem (15분)
- `3_advanced.py` - 🔴 도전 Problem (10분)
